1. SNIFFER

Brief:

This program is a network traffic analyzer written in Python using the Scapy library. 
It monitors HTTP and TLS connections, capturing packets from either a live network interface or a tracefile. 
The parse_https_tls_callback function extracts information such as timestamps, source and destination IPs, ports, HTTP methods, hosts, paths, SNI, and TLS versions from packets containing HTTP or TLS layers. 
The main function main sets up packet capturing based on command-line arguments, allowing flexible monitoring of network traffic for analysis or security purposes.

2. ARP WATCH


Brief:

This Python script detects ARP cache poisoning attacks by monitoring ARP traffic on a specified network interface. 
It uses Scapy for packet analysis. 
The arpwatch function initializes the ARP cache, sets up a callback function to monitor ARP packets, and captures packets using Scapy's sniff function. 
The main function parses command-line arguments and starts ARP monitoring. When ARP packets are detected, the script compares source IP and MAC addresses with the current ARP cache to detect changes indicative of ARP cache poisoning attacks. 
The monitoring continues until interrupted by the user.
